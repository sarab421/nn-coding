# Code Assistant Frontend

A modern React frontend for the Code Assistant API.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The frontend will be available at `http://localhost:3000`.

## Features

- Clean, modern UI with Material-UI
- Dark theme
- Code generation form
- Real-time code display
- Error handling
- Loading states

## API Integration

The frontend connects to the backend API running at `http://localhost:8000/api/generate_code`.

## Dependencies

- React
- Material-UI
- Axios
- React Router DOM
- Emotion
